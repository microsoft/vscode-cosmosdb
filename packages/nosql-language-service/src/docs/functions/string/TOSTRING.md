# TOSTRING

**Category:** String
**Syntax:** `TOSTRING(expression)`

Returns the string representation of the specified value.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `expression` | any | The value to convert to string. |

## Return Value
Returns a string.
