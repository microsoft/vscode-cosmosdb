# IS_FINITE_NUMBER

**Category:** Type check
**Syntax:** `IS_FINITE_NUMBER(expression)`

Returns a Boolean indicating if the type of the value is a finite number (not NaN or Infinity).

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `expression` | any | Any expression. |

## Return Value
Returns `true` or `false`.
