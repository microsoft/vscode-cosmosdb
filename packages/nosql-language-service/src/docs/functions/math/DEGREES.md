# DEGREES

**Category:** Math
**Syntax:** `DEGREES(radians)`

Returns the corresponding angle in degrees for an angle specified in radians.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `radians` | numeric | Angle in radians. |

## Return Value
Returns degrees (numeric).
