# LEFT

**Category:** String
**Syntax:** `LEFT(string, length)`

Returns the left part of a string with the specified number of characters.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `string` | string | The source string. |
| `length` | integer | Number of characters to take. |

## Return Value
Returns a string.
