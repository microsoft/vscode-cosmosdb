# IS_DEFINED

**Category:** Type check
**Syntax:** `IS_DEFINED(expression)`

Returns a Boolean indicating if the property has been assigned a value (i.e., is not `undefined`).

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `expression` | any | Any expression. |

## Return Value
Returns `true` or `false`.
