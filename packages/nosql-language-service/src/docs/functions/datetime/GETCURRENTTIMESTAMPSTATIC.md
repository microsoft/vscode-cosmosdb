# GETCURRENTTIMESTAMPSTATIC

**Category:** Date/Time
**Syntax:** `GETCURRENTTIMESTAMPSTATIC()`

Returns the current UTC date/time as a Unix timestamp, constant for the duration of the query.

## Parameters
None.

## Return Value
Returns a numeric value.
