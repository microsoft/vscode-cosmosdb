# ARRAY_MAX

**Category:** Array
**Syntax:** `ARRAY_MAX(array)`

Returns the maximum value among all elements in the specified array.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `array` | array | An array of primitive values. |

## Return Value
Returns the maximum value.
