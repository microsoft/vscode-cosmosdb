# INTMOD

**Category:** Integer math
**Syntax:** `INTMOD(left, right)`

Returns the result of integer modulo (left %% right) as a 64-bit integer.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `left` | integer | Dividend. |
| `right` | integer | Divisor. |

## Return Value
Returns a 64-bit integer.
