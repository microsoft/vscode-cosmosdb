# AND

Logical conjunction operator. Returns `true` when both operands are true.

## Syntax
```sql
expr1 AND expr2
```

## Operator Precedence

| Operator | Priority |
|----------|----------|
| NOT      | 1        |
| **AND**  | **2**    |
| OR       | 3        |
