# IS_ARRAY

**Category:** Type check
**Syntax:** `IS_ARRAY(expression)`

Returns a Boolean indicating if the type of the value is an array.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `expression` | any | Any expression. |

## Return Value
Returns `true` or `false`.
