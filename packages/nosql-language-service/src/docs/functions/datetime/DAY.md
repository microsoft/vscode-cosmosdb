# DAY

**Category:** Date/Time
**Syntax:** `DAY(datetime)`

Returns the day component (1-31) of a datetime string.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `datetime` | string | A datetime string. |

## Return Value
Returns an integer.
