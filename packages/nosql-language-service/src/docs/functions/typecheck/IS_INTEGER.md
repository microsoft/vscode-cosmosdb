# IS_INTEGER

**Category:** Type check
**Syntax:** `IS_INTEGER(expression)`

Returns a Boolean indicating if the type of the value is an integer (INT64).

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `expression` | any | Any expression. |

## Return Value
Returns `true` or `false`.
