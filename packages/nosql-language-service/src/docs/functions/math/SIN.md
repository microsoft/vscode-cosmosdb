# SIN

**Category:** Math
**Syntax:** `SIN(number)`

Returns the trigonometric sine of the specified angle in radians.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `number` | numeric | Angle in radians. |

## Return Value
Returns a numeric value.
