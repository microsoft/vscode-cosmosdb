# ATAN

**Category:** Math
**Syntax:** `ATAN(number)`

Returns the angle, in radians, whose tangent is the specified numeric expression. Also called arctangent.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `number` | numeric | A numeric expression. |

## Return Value
Returns radians (numeric).
