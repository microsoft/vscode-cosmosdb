# NOT

Logical negation operator.

## Syntax
```sql
NOT expr
expr NOT IN (...)
expr NOT BETWEEN a AND b
expr NOT LIKE pattern
```

## Operator Precedence

| Operator | Priority |
|----------|----------|
| **NOT**  | **1**    |
| AND      | 2        |
| OR       | 3        |
