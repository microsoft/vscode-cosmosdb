# IS_STRING

**Category:** Type check
**Syntax:** `IS_STRING(expression)`

Returns a Boolean indicating if the type of the value is a string.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `expression` | any | Any expression. |

## Return Value
Returns `true` or `false`.
