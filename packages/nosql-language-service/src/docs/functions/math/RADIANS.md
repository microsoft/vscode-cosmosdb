# RADIANS

**Category:** Math
**Syntax:** `RADIANS(degrees)`

Returns radians when a numeric expression in degrees is entered.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `degrees` | numeric | Angle in degrees. |

## Return Value
Returns radians (numeric).
