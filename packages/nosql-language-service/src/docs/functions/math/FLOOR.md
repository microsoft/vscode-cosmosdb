# FLOOR

**Category:** Math
**Syntax:** `FLOOR(number)`

Returns the largest integer less than or equal to the specified numeric expression.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `number` | numeric | A numeric expression. |

## Return Value
Returns an integer numeric value.

## Examples
```sql
SELECT FLOOR(4.8)  -- 4
SELECT FLOOR(-4.2) -- -5
```
