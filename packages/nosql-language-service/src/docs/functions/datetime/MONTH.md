# MONTH

**Category:** Date/Time
**Syntax:** `MONTH(datetime)`

Returns the month component (1-12) of a datetime string.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `datetime` | string | A datetime string. |

## Return Value
Returns an integer.
