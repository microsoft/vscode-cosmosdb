# IS_NULL

**Category:** Type check
**Syntax:** `IS_NULL(expression)`

Returns a Boolean indicating if the type of the value is `null`.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `expression` | any | Any expression. |

## Return Value
Returns `true` or `false`.
