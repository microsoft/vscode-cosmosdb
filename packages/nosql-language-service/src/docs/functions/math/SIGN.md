# SIGN

**Category:** Math
**Syntax:** `SIGN(number)`

Returns the sign value (-1, 0, 1) of the specified numeric expression.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `number` | numeric | A numeric expression. |

## Return Value
Returns -1 (negative), 0 (zero), or 1 (positive).
