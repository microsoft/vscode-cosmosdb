# LENGTH

**Category:** String
**Syntax:** `LENGTH(string)`

Returns the number of characters of the specified string expression.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `string` | string | The string expression. |

## Return Value
Returns a numeric value.
