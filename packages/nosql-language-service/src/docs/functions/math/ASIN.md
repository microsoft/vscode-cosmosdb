# ASIN

**Category:** Math
**Syntax:** `ASIN(number)`

Returns the angle, in radians, whose sine is the specified numeric expression. Also called arcsine.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `number` | numeric | Value between -1 and 1. |

## Return Value
Returns radians (numeric).
