# ARRAY_MIN

**Category:** Array
**Syntax:** `ARRAY_MIN(array)`

Returns the minimum value among all elements in the specified array.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `array` | array | An array of primitive values. |

## Return Value
Returns the minimum value.
