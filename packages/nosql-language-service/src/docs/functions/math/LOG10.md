# LOG10

**Category:** Math
**Syntax:** `LOG10(number)`

Returns the base-10 logarithm of the specified numeric expression.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `number` | numeric | A positive numeric expression. |

## Return Value
Returns a numeric value.
