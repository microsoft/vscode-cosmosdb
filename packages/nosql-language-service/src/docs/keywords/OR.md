# OR

Logical disjunction operator. Returns `true` when either operand is true.

## Syntax
```sql
expr1 OR expr2
```

## Operator Precedence

| Operator | Priority |
|----------|----------|
| NOT      | 1        |
| AND      | 2        |
| **OR**   | **3**    |
