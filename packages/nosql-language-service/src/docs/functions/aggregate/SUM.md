# SUM

**Category:** Aggregate
**Syntax:** `SUM(expression)`

Returns the sum of all values in the expression.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `expression` | numeric | A numeric expression. |

## Return Value
Returns a numeric value.

## Examples
```sql
SELECT SUM(c.amount) FROM c
```
