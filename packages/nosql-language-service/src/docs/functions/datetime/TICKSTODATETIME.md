# TICKSTODATETIME

**Category:** Date/Time
**Syntax:** `TICKSTODATETIME(ticks)`

Converts a ticks value to a datetime string.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `ticks` | integer | Number of ticks since Unix epoch. |

## Return Value
Returns a datetime string.
