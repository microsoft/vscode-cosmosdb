# TAN

**Category:** Math
**Syntax:** `TAN(number)`

Returns the tangent of the specified angle in radians.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `number` | numeric | Angle in radians. |

## Return Value
Returns a numeric value.
