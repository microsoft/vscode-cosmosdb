# ARRAY_MEDIAN

**Category:** Array
**Syntax:** `ARRAY_MEDIAN(array)`

Returns the median value among all numeric elements in the specified array.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `array` | array | An array of numeric values. |

## Return Value
Returns a numeric value.
