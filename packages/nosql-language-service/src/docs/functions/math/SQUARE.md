# SQUARE

**Category:** Math
**Syntax:** `SQUARE(number)`

Returns the square of the specified numeric expression.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `number` | numeric | A numeric expression. |

## Return Value
Returns a numeric value (number * number).
