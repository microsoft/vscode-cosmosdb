# IS_DATETIME

**Category:** Type check
**Syntax:** `IS_DATETIME(expression)`

Returns a Boolean indicating if the type of the value is a datetime string (ISO 8601 format).

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `expression` | any | Any expression. |

## Return Value
Returns `true` or `false`.
