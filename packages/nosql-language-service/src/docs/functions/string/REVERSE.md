# REVERSE

**Category:** String
**Syntax:** `REVERSE(string)`

Returns the reverse order of a string value.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `string` | string | The string to reverse. |

## Return Value
Returns the reversed string.
