# YEAR

**Category:** Date/Time
**Syntax:** `YEAR(datetime)`

Returns the year component of a datetime string.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `datetime` | string | A datetime string. |

## Return Value
Returns an integer.
