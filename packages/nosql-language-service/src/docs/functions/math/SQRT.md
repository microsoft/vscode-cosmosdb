# SQRT

**Category:** Math
**Syntax:** `SQRT(number)`

Returns the square root of the specified numeric expression.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `number` | numeric | A non-negative numeric expression. |

## Return Value
Returns a numeric value.
