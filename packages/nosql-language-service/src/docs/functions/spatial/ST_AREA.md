# ST_AREA

**Category:** Spatial
**Syntax:** `ST_AREA(polygon)`

Returns the area of a GeoJSON Polygon expression.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `polygon` | GeoJSON | A GeoJSON Polygon. |

## Return Value
Returns a numeric value (square meters).
