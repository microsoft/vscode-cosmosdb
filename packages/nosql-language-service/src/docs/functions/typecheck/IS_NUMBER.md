# IS_NUMBER

**Category:** Type check
**Syntax:** `IS_NUMBER(expression)`

Returns a Boolean indicating if the type of the value is a number.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `expression` | any | Any expression. |

## Return Value
Returns `true` or `false`.
