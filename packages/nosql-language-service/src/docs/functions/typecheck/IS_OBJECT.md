# IS_OBJECT

**Category:** Type check
**Syntax:** `IS_OBJECT(expression)`

Returns a Boolean indicating if the type of the value is a JSON object.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `expression` | any | Any expression. |

## Return Value
Returns `true` or `false`.
