# COS

**Category:** Math
**Syntax:** `COS(number)`

Returns the trigonometric cosine of the specified angle in radians.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `number` | numeric | Angle in radians. |

## Return Value
Returns a numeric value.
