# PI

**Category:** Math
**Syntax:** `PI()`

Returns the constant value of PI (3.14159265358979...).

## Parameters
None.

## Return Value
Returns the numeric constant PI.

## Examples
```sql
SELECT PI() -- 3.14159265358979
```
