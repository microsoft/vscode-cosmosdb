# EXP

**Category:** Math
**Syntax:** `EXP(number)`

Returns the exponent (e^number) of the specified numeric expression.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `number` | numeric | A numeric expression. |

## Return Value
Returns a numeric value.
