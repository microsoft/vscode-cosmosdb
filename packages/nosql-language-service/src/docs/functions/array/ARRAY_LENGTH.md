# ARRAY_LENGTH

**Category:** Array
**Syntax:** `ARRAY_LENGTH(array)`

Returns the number of elements in the specified array expression.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `array` | array | The array expression. |

## Return Value
Returns a numeric value.
