# ABS

**Category:** Math
**Syntax:** `ABS(number)`

Returns the absolute (positive) value of the specified numeric expression.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `number` | numeric | The numeric expression. |

## Return Value
Returns a numeric value.

## Examples
```sql
SELECT ABS(-5) -- 5
SELECT ABS(c.balance) FROM c
```
