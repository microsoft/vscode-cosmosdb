# ACOS

**Category:** Math
**Syntax:** `ACOS(number)`

Returns the angle, in radians, whose cosine is the specified numeric expression. Also called arccosine.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `number` | numeric | Value between -1 and 1. |

## Return Value
Returns radians (numeric).
