# DOCUMENTID

**Category:** Other
**Syntax:** `DOCUMENTID(document)`

Returns the internal document ID of the specified document.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `document` | identifier | The document alias (e.g. `c`). |

## Return Value
Returns a string.
