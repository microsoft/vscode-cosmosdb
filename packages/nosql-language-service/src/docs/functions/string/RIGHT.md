# RIGHT

**Category:** String
**Syntax:** `RIGHT(string, length)`

Returns the right part of a string with the specified number of characters.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `string` | string | The source string. |
| `length` | integer | Number of characters to take. |

## Return Value
Returns a string.
