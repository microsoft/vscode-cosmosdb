# SETUNION

**Category:** Set
**Syntax:** `SETUNION(set1, set2)`

Returns an array of elements in the union of both sets, with no duplicates.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `set1` | array | The first set. |
| `set2` | array | The second set. |

## Return Value
Returns an array.

## Notes
- Also available as `SET_UNION`.
