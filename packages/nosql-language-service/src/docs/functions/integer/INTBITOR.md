# INTBITOR

**Category:** Integer math
**Syntax:** `INTBITOR(left, right)`

Returns the result of bitwise OR on two 64-bit integers.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `left` | integer | Left operand. |
| `right` | integer | Right operand. |

## Return Value
Returns a 64-bit integer.
