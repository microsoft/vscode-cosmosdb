# GETCURRENTTIMESTAMP

**Category:** Date/Time
**Syntax:** `GETCURRENTTIMESTAMP()`

Returns the current UTC date/time as a Unix timestamp (milliseconds since epoch).

## Parameters
None.

## Return Value
Returns a numeric value.
