# ARRAY_SUM

**Category:** Array
**Syntax:** `ARRAY_SUM(array)`

Returns the sum of all numeric elements in the specified array.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `array` | array | An array of numeric values. |

## Return Value
Returns a numeric value.
