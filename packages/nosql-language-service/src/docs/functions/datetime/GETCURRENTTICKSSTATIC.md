# GETCURRENTTICKSSTATIC

**Category:** Date/Time
**Syntax:** `GETCURRENTTICKSSTATIC()`

Returns the current UTC date/time in ticks, constant for the duration of the query.

## Parameters
None.

## Return Value
Returns a numeric value (ticks).
