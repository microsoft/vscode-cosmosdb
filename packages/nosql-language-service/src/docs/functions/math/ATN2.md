# ATN2

**Category:** Math
**Syntax:** `ATN2(y, x)`

Returns the angle, in radians, between the positive x-axis and the ray from the origin to the point (y, x).

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `y` | numeric | Y coordinate. |
| `x` | numeric | X coordinate. |

## Return Value
Returns radians (numeric).
