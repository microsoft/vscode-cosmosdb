# LOWER

**Category:** String
**Syntax:** `LOWER(string)`

Returns a string expression after converting uppercase characters to lowercase.

## Parameters
| Name | Type | Description |
|------|------|-------------|
| `string` | string | The string expression. |

## Return Value
Returns the lowercase string.
